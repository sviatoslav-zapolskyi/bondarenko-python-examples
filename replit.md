# bondarenko-python-examples

A collection of Python programming examples.

## Project Structure

- `paragraph25/example1.py` - Turtle graphics example that draws a triangle using forward/right commands

## Setup

- Language: Python 3.12
- Dependencies: `python312Full` (includes tkinter for GUI/turtle graphics), `tcl`, `tk`

## Running

The main workflow runs `python paragraph25/example1.py` using VNC output mode (required for GUI/turtle applications).
